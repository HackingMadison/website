Google Gadget

Adopt-a-Hydrant

Adopt-a-hydrant is an open-source project built and shared by Code for America. It encourages citizen engagement by letting individuals sign up as a caregiver for a fire hydrant when the snow storms hit. This Rails app has been customized and deployed here in Madison.

Launch App »

Transit API

An extension to the original SMS application, this API provides a web services wrapper around the Metro's transit data. Access:'

  • real-time arrival estimates for all stops
  • perform geo-spacial queries for Metro stops
  • get location details for all stops
  • find route details
